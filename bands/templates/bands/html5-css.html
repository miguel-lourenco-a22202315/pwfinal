Seletores CSS Utilizados

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para estilizar elementos HTML em uma página da web. Abaixo estão os seletores e combinadores de seletores que foram utilizados neste site:

Seletores CSS

Tipo de Seletor Utilizado Comentário
Seletor Universal (*) O seletor universal (*) seleciona todos os elementos em uma página. Pode ser útil para aplicar estilos globais, como a definição da fonte padrão para toda a página. Foi utilizado neste site para definir a fonte "Roboto" como fonte padrão.
Seletor de Classe (.) O seletor de classe (.) é utilizado para estilizar elementos com uma classe específica. Foi utilizado para estilizar elementos do cabeçalho e botões.
Seletor de Elemento (tag) O seletor de elemento é utilizado para estilizar elementos HTML específicos. Foi utilizado para estilizar títulos (h1), parágrafos (p) e imagens.

Combinadores de seletores

Combinador Utilizado Comentário
Combinador de Filho direto (>) O combinador de filho direto (>) seleciona elementos que são filhos diretos do elemento especificado. Foi utilizado para estilizar os botões dentro do cabeçalho.
Combinador de Irmão Adjacente (+) O combinador de irmão adjacente (+) seleciona o elemento irmão imediatamente após o elemento especificado. Não foi utilizado nesta página.
