API Meteorologia

Esta API permite consultar informações meteorológicas de várias cidades, incluindo previsões diárias e de 5 dias.

Endpoints Disponíveis

  • Lista de Cidades: https://a22202315.pythonanywhere.com/Meteriologia/api/lista_cidades/
  • Previsão Meteorológica de Hoje para uma Cidade: https://a22202315.pythonanywhere.com/Meteriologia/api/previsao_hoje/{cidade_id}/
  • Previsão Meteorológica para os Próximos 5 Dias para uma Cidade: https://a22202315.pythonanywhere.com/Meteriologia/api/previsao_5dias/{cidade_id}/

Lista de Cidade

Retorna uma lista de todas as cidades disponíveis para consultar a meteorologia.

Formato de Resposta JSON

  • nome: Nome da cidade
  • ID: ID da cidade

Previsão Meteorológica de Hoje para uma Cidade

Retorna informações meteorológicas de hoje para uma cidade específica.

Formato de Resposta JSON

  • nome: Nome da cidade
  • temperatura_minima: Temperatura mínima
  • temperatura_maxima: Temperatura máxima
  • data: Data da previsão
  • descricao_tempo: Descrição do tempo
  • precipitacao: Precipitação
  • icone_tempo: Link para o ícone do tempo

Previsão Meteorológica para os Próximos 5 Dias para uma Cidade

Retorna informações meteorológicas para os Próximos 5 dias para uma cidade específica.

Formato de Resposta JSON

  • nome: Nome da cidade
  • temperatura_minima: Temperatura mínima
  • temperatura_maxima: Temperatura máxima
  • data: Data da previsão
  • descricao_tempo: Descrição do tempo
  • precipitacao: Precipitação
  • icone_tempo: Link para o ícone do tempo
