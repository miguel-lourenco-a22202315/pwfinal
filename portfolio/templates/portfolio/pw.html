Python

Descrição: Python é uma linguagem de programação de alto nível, conhecida por sua sintaxe clara e legibilidade. É amplamente utilizada em desenvolvimento web, ciência de dados, automação, e muitas outras áreas. O Django, por exemplo, é um framework web escrito em Python.

Usado na Cadeira: O Python foi usado essencialmente durante as primeiras 3 semanas da cadeira, servindo como uma fase de adaptação introdutória para o Django.

HTML

Descrição: HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ele utiliza uma série de elementos e tags para estruturar o conteúdo, como textos, imagens, links e outros elementos multimídia.

Usado na Cadeira: O HTML foi utilizado durante uma boa parte da cadeira de Programação Web, sendo utilizado para a criação de diversas páginas simples para a sua introdução, bem como todas as apps realizadas ao longo da cadeira, como aplicação bandas, blog, curso.

CSS

Descrição: CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Ele permite controlar o layout, cores, fontes e outros aspectos visuais das páginas web.

Usado na Cadeira: O CSS foi utilizado mais durante a fase final da cadeira de Programação Web, sendo utilizado para a estilização de todas as aplicações desenvolvidas durante o decorrer do semestre, assim como as que foram criadas para o projeto final, como por exemplo as bandas, o blog, o curso e o portfólio.

APIs

Descrição: APIs (Application Programming Interfaces) permitem a comunicação entre diferentes sistemas de software. Elas são utilizadas para integrar e disponibilizar serviços e dados de uma aplicação para outras.

Usado na Cadeira: As APIs foram introduzidas na parte final da cadeira de Programação Web com o objetivo de implementar uma aplicação de meteorologia com a API do IPMA, bem como a criação da nossa própria API do Tempo.

JavaScript

Descrição: JavaScript é uma linguagem de programação amplamente utilizada para criar conteúdo web dinâmico e interativo. Ele pode ser usado para manipular elementos HTML, validar formulários, criar animações e muito mais.

Usado na Cadeira: O JavaScript foi utilizado durante a fase final da cadeira de Programação Web, não tendo sido muito aprofundado, sendo utilizado para a criação de um relógio com data, horas e segundos na aplicação portfólio, bem como a implementação de um modo escuro na mesma.

Estrutura de um Projeto Django

Um projeto Django é organizado em:

  • Projeto: Contém as configurações globais e arquivos de gerenciamento.
  • Aplicações: Módulos independentes que podem ser reutilizados em diferentes projetos.
  • Models: Definem a estrutura dos dados do aplicativo.
  • Views: Contêm a lógica que manipula os dados e renderiza templates.
  • URLs: Roteiam requisições para as views apropriadas.
  • Templates: Arquivos HTML que exibem dados para os usuários.
  • Statics: Arquivos CSS e JavaScript para estilização e interatividade das páginas HTML.
  • Admin: Interface administrativa gerada automaticamente pelo Django para gerenciar dados do projeto.
  • Djson: Gerenciamento de arquivos e respostas JSON no Django.
  • Loader: Utilitário para carregamento de dados estáticos ou dinâmicos em templates.
  • Forms: Manipulação e validação de formulários HTML no Django.
